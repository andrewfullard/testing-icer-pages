    Open OnDemand

    Open OnDemand is an open-source software, based on the Ohio Supercomputer Center's proven "OSC OnDemand" platform. It provides easy web access to HPCC resources with graphical interfaces. The features of the web portal include (but are not limited to):

    • Easy file management
    • Job submission, management and monitoring across different nodes and resource managers
    • Graphical desktop environments and desktop applications (such as Matlab, Jupyter Notebook, RStudio, ... and more)
    • Command-line shell access
    • One-click app icons to launch your favorite GUI applications (This is New !! See Application Icons on Desktop)

    Connection to HPCC OnDemand

    To connect to HPCC OnDemand, visit https://ondemand.hpcc.msu.edu/. It will first redirect to CILogon website for authentication. From here, select "Michigan State University" as the identity provider and click "Log On". This will redirect you to a page where you can log in with your MSU credentials.

    MSU email address and NetID password are required to access HPCC OnDemand. This may prohibits the access of external collaborators. Please take this into consideration when you choose tools provided on HPCC OnDemand for your group projects involving external collaborators. 

    After sign in, you reach the main page of HPCC OnDemand:

    From the menu on the top of the page, users can select the functions:

    Files

    All user's files in the network file systems (home, research, scratch or software file space) can be accessed. If the permission is provided, download, upload, move, view or edit files in this selection.

    • Go To: you can directly put the location of the folder without navigation.
    • Open in Terminal: It opens a dev-node terminal.
    • New File: It creates an empty file. You can set the name of file when you click 'New file' button.
    • New Dir: It creates a directory.
    • Show Dotfiles: It shows/hides hidden files. By default, it is off.
    • Show Owner/Mode: It shows/hides the owner of files and access modes. By default, it is off.
    • View: If the selected file is ASCII format, It shows the contents of the file. If a directory is selected, it shows the structure, and files' basic information under the directory.
    • Edit: It edits a file contents.

    Jobs

    For Active Jobs, see a list of user's jobs in queue where users can look into job information, monitor or manage the jobs.

    In Job Composer, a job script can be created with resource requests and command lines or from a job template. Users can choose job scripts to submit jobs.

    • From Default Template: A pre-written minimal template. Users need to fill out all resource request.
    • From Template: Matlab, A set of pre-written multi-cores/node/GPU and VASP templates are available.
    • From Specific Path: A user can load his/her own file (currently has some glitches).
    • From Selected Job: It copies a job script from a selected job on the list.

    Interactive Apps

    In Interactive Desktop, an interactive job needs to be requested. Once your job is launched and starts running, a graphical interface desktop is provided. This is excellent for running GUI applications.

    In other GUI apps, a specified GUI program is launched directly after the requested interactive job starts.

    • MATLAB: 2018a-2020b versions are available.
    • Stata: 15SE and 15MP are available.
    • Jupyter notebook: By default, Python 3.7.2 will be used. You can choose your own Python with checking 'Launch Jupyter Notebook using the Anaconda installation in my home directory'.
    • RStudio: Various versions are available (3.4.3\~ 4.0.0)
    #### *Please make sure you request enough memory with a unit for your session. Otherwise, your interactive session is not able to start or your running processes will be terminated.*

    Development Nodes

    Access a dev node and run command lines as a terminal. There is no graphical display at this moment. For GUI, please use Interactive Apps.

    My Interactive Sessions

    it shows all of your opened interactive jobs information.
